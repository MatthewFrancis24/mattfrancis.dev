# Proxmox Homelab

## Overview

This homelab environment is built to simulate real enterprise infrastructure and test system administration, networking, and security configurations.

## Infrastructure

**Hypervisor**
- Proxmox VE

**Virtual Machines**
- Windows Server (Active Directory)
- Windows Client machines
- Linux test servers

## Purpose

The environment is used to:

- Practice domain administration
- Test Group Policy configurations
- Deploy virtual machines
- Simulate enterprise infrastructure

## Skills Demonstrated

- Virtualization
- Active Directory administration
- Network configuration
- Infrastructure troubleshooting
