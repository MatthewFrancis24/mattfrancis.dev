# Active Directory Lab

## Overview

This lab environment simulates a small enterprise Active Directory infrastructure used for testing identity management, domain administration, and group policy configurations.

The environment is hosted within a Proxmox virtualization platform and allows experimentation with Windows Server administration and domain management.

---

## Infrastructure

**Hypervisor**
- Proxmox VE

**Domain Environment**
- Windows Server Domain Controller
- Active Directory Domain Services
- DNS

**Client Systems**
- Windows client machines joined to the domain

---

## Key Configurations

- Domain controller deployment
- Active Directory Domain Services installation
- DNS configuration
- Domain join for client systems
- Organizational Unit (OU) structure
- User and group management
- Group Policy configuration

---

## Skills Demonstrated

- Active Directory administration
- Domain infrastructure deployment
- Group Policy management
- Windows Server configuration
- Identity and access management
