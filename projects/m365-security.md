# Microsoft 365 Security Hardening

## Overview

This project focuses on improving the security posture of a Microsoft 365 tenant by implementing recommended security configurations and identity protection policies.

The goal is to reduce attack surface, strengthen authentication controls, and improve the organization's Microsoft Secure Score.

---

## Key Improvements

- Secure Score analysis and improvement
- Multi-Factor Authentication configuration
- Identity protection policy implementation
- Conditional access configuration
- Security recommendations review

---

## Security Measures Implemented

**Identity Protection**
- Multi-factor authentication
- Strong authentication requirements
- User risk monitoring

**Access Controls**
- Conditional Access policies
- Access restrictions based on risk

**Tenant Security**
- Secure Score recommendations implemented
- Review of authentication methods
- Hardening of identity configuration
